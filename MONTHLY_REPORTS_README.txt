📊 DOWNLOAD REPORTS BY MONTH

Option 1: Export Current Month
- Click on a month tab
- Click "Export CSV This Month"
- File format: inventory_sales_MONTH_YEAR.csv
- Example: inventory_sales_April_2026.csv

Option 2: Export All Data
- Click "Export CSV All"
- Downloads all sales data from all months
- File format: inventory_sales.csv
